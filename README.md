# Nimma Guru Mentor App

Nimma Guru Mentor App is a mentorship and learning platform designed to connect students with mentors for guidance, skill development, career support, and interactive learning through a simple and user-friendly mobile application.

## Features
- Student and Mentor Login
- Mentor Guidance and Support
- User-Friendly Interface
- Firebase Integration
- Real-Time Data Management
- Learning and Career Support

## Technologies Used
- Android Studio
- Kotlin
- Firebase Authentication
- Firebase Firestore
- XML

## Project Objective
The main objective of this project is to help students connect with mentors for learning, career guidance, and skill improvement through an easy-to-use mobile application.

## How to Run
1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle files
4. Run the application on an emulator or Android device
