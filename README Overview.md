# Nimma Guru Mentor App

## Overview
Nimma Guru Mentor App is a mentorship and learning platform designed to connect students with mentors for career guidance, skill development, and interactive learning through a simple mobile application.

## Features
- Student and Mentor Authentication
- Mentor Guidance System
- Firebase Database Integration
- Real-Time Data Storage
- User-Friendly Interface
- Learning Support Platform

## Technologies Used
- Kotlin
- Android Studio
- Firebase Authentication
- Firebase Firestore
- XML

## System Requirements
- Android Studio Hedgehog or above
- Android Device / Emulator
- Internet Connection

## Installation Steps
1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle files
4. Connect Firebase
5. Run the app on emulator or Android device

## Project Structure
- `app/` → Main application source code
- `gradle/` → Gradle configuration files
- `firebase/` → Firebase-related configuration
